# 하늘동물


### 하늘 동물의 특성

+ 각각의 동물은 고유한 특성을 가진다
+ 동물들을 3마리까지 세이브 할 수있다.
+ 세이브 된 동물의 관계에 따라 보너스 점수가 달라진다.
+ 총 다섯종류의 하늘 동물이 있다.



### 하늘 동물의 종류

1. 비둘기

+ 설명: 흔한 새입니다. 어디서나 볼 수 있기 때문에 특별한 능력은 없습니다.

+ 능력: 특별한 능력은 없다. 모든 장애물을 피해가야 한다.


2. 제비


+ 설명: 행운의 새입니다. 이 새를 가지면 행운이 따를 것입니다. 그만큼 보기 드물다.

+ 능력: 동물을 탄 순간부터 5초간 무적의 상태가 됩니다. 무적상태에서는 모든 장애물의 피해를 받지 않습니다. 또한 10초간 점수가 2배로 쌓입니다.


3. 송골매

+ 설명: 가장 빠른 새이다. 빠른 만큼 컨트롤 하기 쉽지 않다.

+ 능력: 스피드가 가장 빠르다. 또한 포식자이니 만큼 비둘기, 제비를 먹을 수 있다.


4. 흑기러기(철새)

+ 설명: 먼 거리를 이동하는 새이다. 겨울철새이므로 수년 킬로미터를 움직인다.

+ 능력: 철새이므로 오랫동안 비행이 가능하다. 하지만 스피드는 지구력 만큼 좋지는 않다.


5. 독수리

+ 설명: 조류 중의 최강의 포식자이다. 모든 동물들 위에 있다고 생각하면 된다.

+ 능력: 모든 동물들을 잡아 먹을 수 있다. 하지만 오랫동안 탈 수는 없다. 지구력이 좋지 않다.


### 하늘 동물들간의 케미

+ 각각의 동물들의 조합마다 보너스 점수가 부여된다
