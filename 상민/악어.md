# 정글

### 악어

+ 업그레이드

  1. 악어를 탈 때 물 속에서 이동 속도 증가

  2. 악어가 다른 악어를 사냥 가능

  3. 다른 동물을 타고 있을 때 악어의 공격 속도 저하

  4. 악어의 티켓 수익 50% 증가

  5. 동물 사냥 시 악어의 행복 상태 더 오래 지속

  6. 악어가 무엇이든 사냥 가능

  7. 악어를 타고 다른 동물 사냥하면 보너스 동전 휙득

  8. 스탬피드 중에 희귀 악어의 출현 빈도 2배증가

  9. 스탬피드에 새로운 희귀 악어 추가
